# pongtard-server
Servidor del joc de pong amb controlers remots. Creat amb unity 3d creat amb Node.JS

### Instalacio:
```sh
npm i
```

### Arrancar server:
```sh
node pongserve.js
```

### Serveis disponibles:
http 3000

socket 3010
